#RunCli
Command Line Interface.
    `migrate, seed, generate Eloquent ORM migrations, generate resources`

## Install
    add to composer
    "require": {
        "runcmf/runcli":  "dev-master"
      },

## Usage
    php cli - for help
    php cli make:seed vendor/runcmf/runbb
    php cli make:migrate vendor/runcmf/runbb
